## Getting Started

### Prerequisites

- Yarn or Npm

### Installation

1. Clone the reop

`git clone https://github.com/brunoeduardodev/react-redux-auth-setup.git`

2. Install NPM packages

`npm install`

or

`yarn install`

3. Run the project in development mode

`yarn start`

4. Build project:

`yarn build`
